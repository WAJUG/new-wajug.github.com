# wajug.github.com
The website of WAJUG.
Based on the courtesy work of eyaka.

## To test/run the site

* With Docker:
```
docker run --rm -p 4000:4000 -v $(pwd):/srv/jekyll -ti jekyll/jeky jekyll serve
```
* OR
  - Install jekkyl.rb
  - Run jekyll serve

* Launch your browser on http://localhost:4000/
